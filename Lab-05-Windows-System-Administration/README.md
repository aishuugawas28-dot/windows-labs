# Lab 05 - Windows System Administration

##  Objective

The objective of this lab is to learn the basic Windows administration tools used by IT Support Engineers to manage, monitor, and troubleshoot Windows systems.

---

##  Lab Environment

- **Operating System:** Windows 11 Pro
- **Virtualization Platform:** Oracle VirtualBox
- **PC Name:** ITSUPPORT-01

---

##  Tasks Performed

### 1. Renamed the Computer
- Renamed the virtual machine to **ITSUPPORT-01**.

---

### 2. Device Manager

Verified the following hardware devices:

- Display Adapter
- Disk Drives
- Network Adapter
- USB Controllers
- Processors

**Observation**

- All major devices were functioning properly.
- A **Base System Device** with a yellow warning icon was detected under **Other Devices**, indicating a missing driver that requires further investigation.

---

### 3. Disk Management

Opened **Disk Management** and verified:

- EFI System Partition
- Windows (C:) Partition
- Recovery Partition
- CD/DVD Drive

**Observation**

Disk structure was healthy and correctly configured.

---

### 4. Windows Services

Verified the status of the following services:

| Service | Status | Startup Type |
|---------|---------|--------------|
| DHCP Client | Running | Automatic |
| Print Spooler | Running | Automatic |
| Windows Update | Manual | Trigger Start |

---

### 5. System Information

Verified:

- Operating System
- Windows Version
- BIOS Mode
- Processor
- Installed RAM
- System Manufacturer
- System Model
