# Lab 04 - Windows Troubleshooting

## Objective

Learn basic Windows network troubleshooting using Command Prompt.

## Environment

- Oracle VirtualBox
- Windows 11 Pro

## Commands Used

```cmd
ipconfig
ipconfig /all
ping 8.8.8.8
ping google.com
nslookup google.com
```

## Observations

- Ethernet Adapter was connected.
- IPv4 connectivity was working.
- Internet connectivity was available.
